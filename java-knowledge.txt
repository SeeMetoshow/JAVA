1. 关键字--keyword -- java中具有特殊含义的字符串.
   例如class、public、int等等, 注意:关键字是小写的.
2. 标识符--identifier -- java中任何需要命名的字符序列都称为标识符.
   如:变量名、类名、方法名、接口名、包名...
   
   命名规则:
   (1)可包含26个英文字母的大小写, 0-9, '_'或'$'.
   (2)数字 不可以作为开头.
   (3)不可以使用关键字和保留字, 但可以包含有关键字和保留字.
   (4)java中严格区分大小写, 并且对于长度没有限制.
   (5)标识符不能包含空格.
								规则需要严格遵守.	
	命名规范:
   (1)常量名: 所有字母都要 大写 , 多单词 时, 每个单词用 下划线'_' 连接. 如:XXX_YYY.
   (2)变量名、方法名: 由 多单词 组成时, 第一个单词的 首字母 小写, 
					  第二个单词开始每个单词 首字母 大写, 如: xxxYyyZzz. -- 小驼峰
   (3)类名、接口名: 多单词 组成时, 所有单词的首字母都要 大写, 如: XxxYyyZzz.
   (4)包名: 多单词 组成时, 所有字母都要 小写.
					规范可以不遵循,但是为了与己方便,与人方便,最好遵守.
	一点要求:
			命名时要做到"见名知意".
3. 定义变量的格式: 变量类型  变量名  =  变量值;
   可以先声明变量, 然后再给变量赋值.
   注意: 变量作用域的问题.
4. 变量的数据类型
	基本数据类型:
		整型: int \ short \ byte \ long 注意: 声明long类型变量时, 必须以 “L” 、"l" 结尾.
		浮点型: float \ double			注意: 声明float类型的变量时, 必须以"F" 、"f" 结尾.
		字符型: char
		布尔型: boolean
	引用数据类型:
		类: class
		接口: interface
		数组: array
